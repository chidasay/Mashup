# Mashup

概要  
WebAPIを用いて、おすすめの本と現在地近くの本屋を検索し、表示するプログラムです。  
外部APIの稼働状況によっては、うまく動作しない場合があります。ご了承ください。  
  
使用技術  
Java/楽天RSS/OpenStreetMap/OverpassAPI/
